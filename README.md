# Computer-Based Lottery System

A web-based lottery system developed as a Bachelor's 
graduation project at the European University of Lefke 
(2022–2023).

## Features
- User registration and authentication
- Lottery contest entry system
- Payment processing
- Admin dashboard for managing contests and winners
- Winner selection and announcement

## Technologies
PHP | MySQL | HTML | CSS | JavaScript

## Database
MySQL database with user management and contest records
